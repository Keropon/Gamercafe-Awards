# GC Awards 2022

Formulario utilizado para la encuesta anual de la comunidad de [Gamercafe](https://www.gamercafe.cl).

## Setup

```sh
npm install
```

### Compilación para Desarrollo

```sh
npm run dev
```

### Compilación para Producción

```sh
npm run build
```

### [ESLint](https://eslint.org/)

```sh
npm run lint
```
